주문력 마법사로 분류된 챔피언들

추천 3신기

피바라기(피바) : 공 15%, 주문력 15, 흡혈 20%, 마저20. 전투당 한 번 체력의 25%에 해당하는 보호막 5초 망토 + 대검

거인의 결의(거결) : 방 20 공속 10%. 공격 하거나 받을 때 마다 공2%, 주문력 2(최대 25회) 최대 중첩시 방마저 20. 조끼 + 곡궁

스테락의 도전(스테락) : 체력 150, 공 15%. 체력 40%가 까이면 최대체력 25%, 공 35% 증가. 대검 + 벨트

밤의 끝자락(밤끝) : 공 10%, 방어력 20. 체력이 40% 까였을 때 지정불가(무적), 해로운 효과 제거, 이후 공속 15% 증가 대검 + 조끼

무한의 대검(인피, 무대) : 공 35%, 치명타 35%. 스킬에 치명타 적용 가능. 대검 + 장갑

정의의 손길(정손) : 공 15%, 주문력 15, 흡혈 15%. 두 효과중 하나가 무작위로 2배가 됨. 마나 15, 치명타 20%. 눈물 + 장갑

예외적으로 워윅, 올라프 같은 평타형 공격력 전사에게만 좋은 아이템

구인수의 격노검(구인수) : 주문력 10, 공속 10%. 공격할 때 마다 공속 5%. 곡궁 + 지팡이

수은(퀵실) : 마저 20, 공속 30%, 치명타 20%. 14초동안 CC면역, 이 시간동안 2초마다 공속 4%. 올라프는 필요없음. 망토 + 장갑
