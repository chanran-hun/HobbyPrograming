주문력 전사로 분류된 챔피언들

추천 3신기

이온충격기(이온) : 주변적의 마법저항력을 감소시키고, 상대가 스킬을 쓸 때 본인이 일정 수준의 반사 피해를 입게 한다. 망토 + 지팡이

피바라기(피바) : 모든 피해 흡혈을 얻는다, 일정 수준으로 체력이 깎이면 보호막을 받음. 대검 + 망토

크라운 가드 : 전투 시작시 보호막, 방어막이 제거되면 주문력 증가. 조끼 + 지팡이

보석건틀릿(보건) : 스킬에 치명타가 적용될 수 있음, 스킬 데미지를 더 향상시켜준다. 지팡이 + 장갑

대천사의 지팡이(대천사) : 5초마다 주문력 30 증가, 오래 버틸수록 세짐. 지팡이 + 눈물

정의의 손길(정손) : 모든피해 흡혈, 보건과 시너지가 좋음(치명타 확률증가). 눈물 + 장갑

거인의 결의(거결) : 공격 하거나 공격 받을 때 마다 주문력 증가(2씩 최대 25회), 최대 중첩시 방마저 20. 조끼 + 곡궁

라바돈의 죽음모자(라바돈) : 주문력 +50, 피해량 증가 15%. 데미지 하나는 보장해줌. 지팡이 + 지팡이
